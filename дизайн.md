# UI Design Spec (Figma) — Облачный AI-SaaS для мебельных фабрик (RU, 2025)

## 0. Дизайн-принципы (2025)
- Простота и масштабируемость: чёткая иерархия, высокий контраст, минимум скрытых действий. :contentReference[oaicite:1]{index=1}
- Современная выразительность: аккуратные 3D/стеклянные панели, мягкие тени (без перегруза), «bento-grid» на дашбордах. :contentReference[oaicite:2]{index=2}
- Основа — дизайн-система с токенами (цвет/типографика/радиусы/тени) и единым источником правды. :contentReference[oaicite:3]{index=3}
- AI-by-default: встроенные ассистенты, «объяснимость» решений, Dev Mode/MCP для связи с кодом. :contentReference[oaicite:4]{index=4}

---

## 1. Информационная архитектура (IA)
**Разделы:**
1) Главная (Дашборд); 2) Заказы; 3) Спецификация (BOM); 4) Фурнитура (RAG-подбор); 5) CAM (DXF/G-code); 6) Интеграции (1С); 7) Журналы (Аудит/Ошибки); 8) Настройки (Пользователи/Токены/Токены-дизайна).

**Навигация:** левое док-меню (иконка + лейбл), «действие дня» — primary CTA в app bar; глобальный поиск (⌘/Ctrl-K) с AI-подсказками.

---

## 2. Каркас страниц (Figma Frames)

### 2.1 Dashboard (Desktop 1440×900, 12-колонок, gutter 24, margin 24)
- Header: логотип, глобальный поиск (AI), CTA «Новый заказ».
- Bento-грид карточек (Auto Layout): 
  - «Загрузить ТЗ/эскиз» (primary), 
  - «Последние заказы» (таблица 5 строк), 
  - «Очередь CAM» (статусы), 
  - «Качество AI» (метрики golden-set).
- Bottom strip: «Гид по началу работы» (микро-онбординг 3 шага).
> Bento-grid и выразительные элементы соответствуют трендам 2025. :contentReference[oaicite:5]{index=5}

### 2.2 Заказ (Wizard, 4 шага)
1) ТЗ: textarea + загрузка эскиза (dropzone) → OCR прогресс.
2) Извлечённые параметры (editable form) → валидации/единицы (мм).
3) Предпросмотр BOM → экспорт/сохранить черновик.
4) Запуск CAM → уведомление/вебхук → ZIP.

### 2.3 Фурнитура (RAG-подбор)
- Панель фильтров (тип/толщина/нагрузка/бренд).
- Результаты: карточки с «Причина выбора» (AI-объяснение) + значки совместимости.
- Сайдбар «Источники» (SKU/версия/релевантность) — прозрачность AI.

### 2.4 CAM
- Параметры раскроя/фрезеровки (правый сайдбар).
- Очередь заданий (статусы, ETA, retry).
- «Скачать ZIP» (DXF, NC, спецификация, отчёт качества).

### 2.5 Интеграции
- Карточки: 1С (OData), каталоги фурнитуры (импорт), вебхуки.
- Маппинг полей (табличный редактор с подсветкой несовпадений).

## 3. Дизайн-система (Foundations → Components → Patterns)

### 3.1 Foundations (Design Tokens)
| Токен | Ключ | Значение/Правило |
|---|---|---|
| База цвета | `color.neutral.0…1000` | Пять нейтральных уровней для светлой/тёмной темы |
| Акцент | `color.primary` / `color.secondary` | Material 3 подход к палитрам/семействам оттенков |
| Радиусы | `radius.sm/md/lg/2xl` | 4, 8, 12, 16 px |
| Тени | `shadow.sm/md/lg` | y=2/6/12, blur=6/16/28, opacity ≤12% |
| Типо-масштаб | `type.display/h1…body/sm` | Модульная шкала 1.20 (display 54 → body 14–16) |
| Пробелы | `space.2…40` | Шаг 2/4/8 px; базовый 8 px |
| Анимация | `motion.fast/normal/slow` | 120/200/320 ms; easing стандарт/overshoot |

> Токены Material 3 и их уровни документированы (reference/semantic component tokens). :contentReference[oaicite:6]{index=6}  
> Тренд 2025 — зрелые дизайн-системы с токенами и строгой документацией. :contentReference[oaicite:7]{index=7}

### 3.2 Цветовые темы
- **Light**: минимализм, высокий контраст (AA/AAA), приём «стеклянных» панелей — прозрачно ≤8%, blur деликатный.  
- **Dark**: «expressive» Material 3 — насыщенные акценты, скругления, упругие анимации. :contentReference[oaicite:8]{index=8}  
> Неоморфизм/глассморфизм — дозировано, с проверкой доступности. :contentReference[oaicite:9]{index=9}

### 3.3 Типографика
- Variable font (кириллица, цифры табличные), базовый размер 16 px (табличные интерфейсы), интерлиньяж 1.4, дисплей-стили для дашборда.

### 3.4 Компоненты (Auto Layout + Variants)
Buttons, Inputs, Selects, Tabs, Tables (row density sm/md), Cards (bento), Breadcrumbs, Modals/Drawers, Toasts, Empty States, Upload, Progress, Stepper, Badge, Tag, KPI Tile, Code/Log viewer.

### 3.5 Паттерны
- **Wizard**: 4 шага (ТЗ→Параметры→BOM→CAM).  
- **RAG-подбор**: выдача с объяснениями и ссылками на источники.  
- **Queue**: статусы, повтор, фильтры по ошибкам/клиентам.

## 4. Гайд по движению (Motion)
- Переходы страниц: 180–220 ms, ease-in-out.  
- Подтверждения/успех: 160 ms + лёгкий scale (1.02).  
- Перетаскивание карточек (bento): spring 220–260 ms.  
- Skeletons на загрузках LLM/OCR, прогресс-бар на CAM.

> «Expressive» обновления Material 3 усиливают динамичность, но приоритет — функциональность. :contentReference[oaicite:10]{index=10}

---

## 5. Доступность и продуктивность
- Контраст ≥ 4.5:1 (текст/фоны), фокус-стили видимы.  
- Кликабельная зона ≥ 44×44 px; таб-навигация; ARIA для таблиц/степпера.  
- Горячие клавиши: глобальный поиск (⌘/Ctrl-K), «Сохранить» (⌘/Ctrl-S), «Запустить CAM» (⇧+Enter).  
- Валидации «на лету», единицы — всегда «мм» в явном виде.

## 8. Контент-правила
- Единицы измерения: **мм**; маски чисел; подсказки примеров (placeholder).  
- Язык системных сообщений — простой, без жаргона; ошибки с рекомендацией исправления.  
- Плейсхолдеры: реальные примеры шкафов/фасадов.

---

## 9. Контроль качества UI
- Чек-лист доступности (контраст, фокус, таб-порядок).  
- Снимки/перегоны для «bento» и теней в тёмной теме (не превышать opacity).  
- Табличная плотность: sm (компакт), md (стандарт); не использовать xs.
